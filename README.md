## 1、项目介绍

选课管理系统1拥有两种角色，功能简单，实现了管理员对课程的增删改，和学生选课功能


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：Bootstrap、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可）
- maven项目：否 
- 硬件环境：Windows 或者 Mac OS
